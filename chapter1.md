# Gradle标准库

---

以下依赖在基础库中已经引用，在项目中不用重复引用

```
compile 'com.github.bumptech.glide:glide:latest.release' //加载图片
compile 'com.google.code.gson:gson:latest.release' //json解析
compile 'com.alibaba:fastjson:latest.release.android' //json解析
compile 'com.scwang.smartrefresh:SmartRefreshLayout:latest.release' //上拉加载更多，下拉更新
compile 'com.mylhyl:acp:latest.release' //权限管理
compile 'com.lzy.net:okgo:latest.release' //网络加载框架
compile 'com.jakewharton:butterknife:8.8.1' //自动取出控件，省去findViewById操作
compile 'com.jakewharton:butterknife-compiler:8.8.1' //自动取出控件，省去findViewById操作
implementation 'com.jude:swipebackhelper:latest.release' //右滑关闭当前页面
compile 'com.orhanobut:logger:latest.release' //日志
compile 'com.qianwen:update-app:latest.release' //APP更新
compile 'com.qianwen:okhttp-utils:latest.release' //okhttp工具类，版本更新用
```
